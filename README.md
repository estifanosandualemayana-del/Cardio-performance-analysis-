# Cardio Performance Analysis

This project focuses on analyzing cardiovascular health data to understand how different factors affect overall performance and condition.

I worked on this as part of my data science learning journey, trying to practice data cleaning, exploration, and drawing insights from real-world data.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## What I Did
- Cleaned and prepared the dataset
- Explored relationships between features like age, weight, and blood pressure
- Visualized key patterns using charts
- Identified trends related to cardiovascular condition

## Key Insights
- Some features show clear patterns with cardio condition
- Lifestyle-related factors seem to influence performance
- Data visualization helped in understanding hidden trends

## Dataset
The dataset is related to cardiovascular health and is commonly used for analysis practice on Kaggle.
